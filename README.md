# Overview

Pokédex, a high technical device, designed to record and store data on all species of Pokémon.
In this web application, the user can search for their favorite Pokémon and observe their abilities, health bars and evolutions!
This website was built with pure HTML, CSS and Vanilla JavaScript for practice and to expand my knowledge in Web Development.

# Features

Here are the features the Pokédex offers:

- Search for a Pokémon by name
- View their heights & weights
- View their health bars
- Explore their abilities
- Explore their evolutions
- Dark mode
- Light mode
- Sound/volume effect

# Installation
To run this project, you must clone this repository to your local computer by typing:
``git clone https://github.com/schnattiba1/pokedex.git``

# Usage
Once this repository has been cloned, you can sucessfully run this project. There are two ways to run this project:

Option 1:
 - Open your folder 
 - Navigate to the pokédex folder
 - Right click on "index.html" file
 - Select "open with" and choose the browser you would like to open

Option 2:
  - Open your VSCode
  - Open your folder
  - Select the pokédex folder
  - Click on "select folder"
  - If "go live" has been installed, click on it
